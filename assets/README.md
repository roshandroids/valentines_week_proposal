# Assets Guide

## Required Assets

### Lottie Animations (JSON format)

Download from [LottieFiles](https://lottiefiles.com/):

1. **day1_rose.json** - Rose/flower animation
2. **day2_teddy.json** - Teddy bear animation
3. **day3_chocolate.json** - Chocolate/candy animation
4. **day4_propose.json** - Proposal/ring animation
5. **day5_hug.json** - Hug/embrace animation
6. **day6_kiss.json** - Kiss/lips animation
7. **day7_valentine.json** - Heart/valentine animation
8. **countdown.json** - Lock/clock animation for countdown screen

### GIFs

Download romantic/cute GIFs from [Giphy](https://giphy.com/) or [Tenor](https://tenor.com/):

1. **day1.gif** - Rose day themed
2. **day2.gif** - Teddy bear themed
3. **day3.gif** - Chocolate themed
4. **day4.gif** - Proposal themed
5. **day5.gif** - Hug themed
6. **day6.gif** - Kiss themed
7. **day7.gif** - Valentine themed

**Note:** Keep file sizes small (<500KB each) for optimal web performance.
